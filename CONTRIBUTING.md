# Contributing to nvllm

## Current Status: Contributions Closed

nvllm is a young fork and we're still stabilizing the codebase. We are **not accepting pull requests or feature requests** at this time.

### What you CAN do

- **Report bugs** — If something doesn't work on your GB10/GH200, open an issue with your error log
- **Ask questions** — Use GitHub Discussions (if enabled) or Issues tagged `question`
- **Share results** — We'd love to hear your performance numbers

### What we're NOT accepting right now

- Pull requests (they will be closed without review)
- Feature requests
- Refactoring suggestions
- CI/CD contributions

### When will contributions open?

Once we've validated the fork against multiple vLLM upstream releases and established a stable rebase workflow, we'll open contributions. Watch the repo for announcements.
